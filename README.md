## 🎯 Goal

* Understand concepts deeply (not just memorize)
* Practice problem-solving regularly
* Track my progress over time

## 🧠 What I’m Focusing On

* Time complexity (Big-O)
* Core data structures (arrays, linked lists, hash tables, trees, etc.)
* Algorithm design techniques
* Problem-solving patterns

---

## ✍️ How I Study

For each topic or lecture:

* I rewrite concepts in simple language
* I avoid copying directly from sources
* I try to explain things as if I’m teaching myself
* I connect concepts to problems

---

## 💻 Practice Approach

After learning a concept:

* Start with simple problems
* Understand the brute-force solution
* Improve it step by step
* Focus on *why* the solution works
